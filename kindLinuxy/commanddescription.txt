경로이동
리눅스 명령어 cd와 상응되는 명령어이다.
인자로 특정 경로(상대 경로)를 입력받으며, 현재 경로를 인자로 입력받은 경로로 변경한다.
목록
리눅스 명령어 ls와 상응되는 명령어이다.
현재 경로에 있는 파일과 폴더의 목록을 출력한다.
옵션을 선택하면 옵션에 따라 형식이나 정렬 기준을 변경하여 출력한다.
폴더생성
리눅스 명령어 mkdir와 상응되는 명령어이다.
인자로 폴더 명을 입력 받으며, 인자로 받은 이름을 가진 폴더를 현재 경로에 새로 생성한다.
폴더삭제
리눅스 명령어 rmdir와 상응되는 명령어이다.
인자로 폴더 명을 입력 받으며, 인자로 받은 이름을 가진 폴더가 현재 경로에 존재하면
 폴더 안에 하위 파일이 존재하지 않을 경우 해당 폴더를 삭제하고, 
인자로 받은 이름을 가진 폴더가 현재 경로에 존재하지 않거나 폴더 안에 하위 파일이 존재하면 오류 메시지를 출력한다.
설명
인자가 없을 경우 <친절한 리눅씨>에서 지원하는 모든 명령어와 그에 대한 설명을 출력하며,
인자가 있을 경우 리눅스 명령어 'man [명령어]' 에 상응되는 명령어를 실행하여
 현재 프로그램에 존재하는 명령어에 대한 설명을 한글로 출력한다.
